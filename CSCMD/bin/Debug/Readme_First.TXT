!Desktop OS Version Build 4300 XMAS Edition 

Damit du !DesktopOS ( Arbeitstitel ) in der bestmöglichen experience erleben kannst,
bitte ich dich die Auflösung deines Monitors auf 1366x768 Bildpunkte einzustellen.

Zukünfitge Version von !DesktopOS sollten eine saklierbare Auflösungsmatrix im Programmcode implementiert bekommen.


Warum XMAS Edition?


Die XMAS Edition von !DesktopOS hat kurz vor Jahresende einige Bugfixes bekommen und einige neue Visual Designs.

Als "Easter-Egg" ist in !DesktopOS die "XMAS Happy Christmas 2020" APP integriert.
Starte sie einfach in der !Desktop Terminal Oberfläche durch Eingabe von '/xmas'.

Möchtest du den Terminal wieder verlassen '/exit' oder auf "Applications" klicken und dann auf "Back to !DesktopOS"


Vielen Dank und viel Spa0 mit !DesktopOS.
